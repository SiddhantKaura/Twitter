# Requirements

- Post a tweet.
  - [Tweet cannot have more than 250 chars]
  - [Tweet should support image upload / video upload]

- Any post should be visible to all users who follow the author.

- Followers/Author can Comment / like on a tweet or on comment.

- Retweeting

- User Profile:
  - Name
  - Follower count
  - Bio
  - Last 10 tweets from user

- Pagination on tweets
- User auth

- Every tweet might be having a hashtag

- Mentioning
